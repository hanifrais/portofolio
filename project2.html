About the Project

This project focuses on visualizing sales data for better decision-making. Key features include:

  • Data from over 300,000 sales transactions.
  • Interactive visualizations created using Tableau.
  • Insights on revenue, customer segments, and sales trends.

Key Challenges

Addressed the following challenges during the project:

  • Data cleaning and preparation to handle missing values.
  • Optimizing dashboard performance for large datasets.

Results

The dashboard improved sales reporting efficiency by 50% and helped the management identify key growth areas.
